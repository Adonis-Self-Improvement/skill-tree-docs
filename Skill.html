    Skill constructor

    Parameters:
    Name Type Description
    id
    title string

    Skill title (READING)
    level number

    Skill level (3)
    goal string

    The success condition for the skill to be complete
    frequency number

    The frequency at which the task of the skill must be completed. Say you have to do x thing 3 times a week, 3 would be the frequency, and weekly would be the time interval.
    interval string

    The time interval of the skill. If a skill was to be done x times weekly, weekly would be the interval. Valid values would be "day", "week", "month", "year", etc.
    timelimit number

    The number of days for which you need to maintain the skill before acquiring it
    xp number

    The amount of XP granted upon completion of the skill
    icon string

    image path for the icon to be displayed, relative to "/icons/" folder
    requires

    required skills
    children

    child skills
