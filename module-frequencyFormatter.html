    static getDaysBetweenDates()
    modules/dateHelper.js, line 101

    Get the number of days between two dates - not inclusive.

    "between" does not include the start date, so days between Thursday and Friday is one, Thursday to Saturday is two, and so on. Between Friday and the following Friday is 7.

    e.g. getDaysBetweenDates( 22-Jul-2011, 29-jul-2011) => 7.

    If want inclusive dates (e.g. leave from 1/1/2011 to 30/1/2011), use date prior to start date (i.e. 31/12/2010 to 30/1/2011).

    Only calculates whole days.

    Assumes d0 <= d1
